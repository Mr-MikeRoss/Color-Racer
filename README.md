# Door-to-Colors
## Description
This application will present the user with a form that asks a series of questions like, "First Name or Business name" and "Content you'd like to include." Page will then load a default html structure with the content updated based on information entered into the form and save that information locally. Several buttons will be displayed with names of different styles and when the user clicks on the button it will update the style of the page. This application will contain multiple styles the user can choose.
